Self-Hosted SaaS Analytics Platform

GA4 + Multi-Agent Simulation platform for acquisition, conversion, and behavioral analysis with automated insights.

Overview

Problem

SaaS teams often rely on disconnected analytics sources, making it difficult to connect user behavior, acquisition channels, and conversion outcomes into one reliable decision system.

Solution

I built a self-hosted analytics pipeline that combines GA4 events with a simulated internal database, then layers multi-agent traffic simulations on top to test realistic behavior and conversion scenarios.

Outcome

  • Unified view of funnel performance and drop-off points from acquisition to conversion.
  • Interactive Streamlit dashboards for channel efficiency and cohort behavior.
  • Automated trend and anomaly detection for faster, data-driven decisions.

Tech Stack

Python SQL GA4 Streamlit Docker Data Pipeline Anomaly Detection

Key Features

  • Integrated GA4 event ingestion with internal simulation-ready storage.
  • Multi-agent simulation framework for user flow, traffic spikes, and conversion behavior modeling.
  • Dashboards tracking funnel drop-offs, channel performance, and cohort movement over time.
  • Automated alerting for trend breaks and anomalies in key business metrics.
  • Containerized deployment of ingestion, processing, and visualization services.
